ubuntu linux setup you first need to run this command - it makes sure you got everything needed installed
sudo apt install python wget git kdenlive youtube-dl -y && echo -e "\n\n\033[32meverything is ready for vidding!"

windows10 setup -install Kdenlive just in case link scroll down a bit to find windows version
-click on start to open the windows10 start menu and start typing powershell till you find powershell then right click on it and choose run as administrator. once the powershell opens up copy bellow command into it and press enter
Enable-WindowsOptionalFeature -Online -FeatureName Microsoft-Windows-Subsystem-Linux

it will install itself. After installing it will prompt you to restart so just restart.
-open microsoft store and search for ubuntu 18.04 LTS

ubuntu linux now run this in a folder you would like to vid guystanding
wget "https://docs.google.com/uc?id=17pv_uV8kLZSfoAgo-EfEXL_MgRo4uTtr&export=download" -O "$PWD/simple plan-last one standing.mp3" && youtube-dl --newline -i -o "$PWD/%(title)s.%(ext)s" --ignore-config --hls-prefer-native "https://www.youtube.com/watch?v=K34j3YQprgk" && wget https://github.com/22i/mykdenliveprojects/raw/master/guystanding.kdenlive && echo -e "\n\n\033[32meverything downloaded! run guystanding.kdenlive and start editing!"

windows10 replace every i1 with your windows 10 username - you can do this by copying below text into notepad and using CTRL+F to find i1 then replace every i1 with your windows10 username
wget "https://docs.google.com/uc?id=17pv_uV8kLZSfoAgo-EfEXL_MgRo4uTtr&export=download" -O "/mnt/c/Users/i1/Downloads/simple plan-last one standing.mp3" && wget https://github.com/22i/mykdenliveprojects/raw/master/guystanding.kdenlive -O /mnt/c/Users/i1/Downloads/guystanding.kdenlive && youtube-dl --newline -i -o "/mnt/c/Users/i1/Downloads/%(title)s.%(ext)s" --ignore-config --hls-prefer-native "https://www.youtube.com/watch?v=K34j3YQprgk" && echo -e "\n\n\033[32meverything downloaded! run kdenlive then open the guystanding.kdenlive project in your downloads folder and start editing!"
