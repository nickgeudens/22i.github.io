How to make your github page like this one

1. visit github you will need to have a github account

2. once logged in create a new repository with name for example if your github username is pico then name it pico.github.io

- in this tutorial pico will be the username example change pico to your github username

3. download my website source code its about 100mb

4. after download is complete extract the content of the zip

5. rename extracted 22i.github.io-master to pico.github.io

6. visit your pico.github.io repository that you created on github

7. in file manager go inside pico.github.io folder

8. select everything there and drag files over to github

9. now just commit everything
